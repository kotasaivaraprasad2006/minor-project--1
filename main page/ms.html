Merg Sort Algorithm

Merge Sort is a divide-and-conquer sorting algorithm that works by recursively splitting the input array into smaller sub-arrays until each sub-array has only one element (which is considered sorted). It then merges the sub-arrays back together in a sorted order. The merging process involves comparing the elements of each sub-array and combining them into a single, sorted array. This process repeats until the entire array is merged and sorted. Merge Sort guarantees a time complexity of O(n log n) in all cases, making it efficient even for large datasets, though it requires additional space for merging.
